Robot Framework Settings

Optional %SUITE will be replaced with the name of the current suite. Note: the current file will not be added to the command. You must give a complete command line for running tests.

The command can include any rflint arguments. For example: rflint --argumentfile mypreferences.args. Note: The current filename will be appended to whatever command you give.

Robotframework-hub provides data for quick-docs and autocomplete.
